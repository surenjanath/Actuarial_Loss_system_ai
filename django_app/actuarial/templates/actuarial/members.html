How Crew configuration works

Crew agents — Each card is one sequential step in the run. Click Edit to set role, goal, and backstory (used directly by CrewAI), optional display label, department, step type (task template), and optional Ollama model (empty = default from Settings → Local LLM). Add or remove agents within the limits below.

Runs & instructions — Whole-run instructions, streaming output, past runs, approval, and PDF live on Crew runs. This page only configures the agent pipeline.

Saved configuration — Pipeline and crew instructions are stored in the database (shared for this app). Configure Ollama under Settings → Local LLM. See AI integrations for a hub of related screens.

Nothing is sent to a cloud LLM unless you run Ollama locally.
